FSYS Tool

Extract and repack Genius Sonority .fsys archives directly in your browser. Everything runs locally — no files leave your computer.

Use this tool to:

  • Extract the individual .dat, .pkx, or other entries from an .fsys archive so you can import them into Blender.
  • Repack a folder of edited entries back into a new .fsys archive that you can drop into your game's files/ tree.

For the full workflow context, see the importer guide or the "importing the model back into the game" section of the exporter guide.

Downloads a single <name>.zip. Unpack it to get a folder containing every entry file alongside <name>-manifest.json — that folder is what the Repack tab expects.

Drop the extracted folder here, or use the picker below.

Pick (or drop) the folder that contains <name>-manifest.json. Each entry's file is resolved by name from inside the same folder. The rebuilt .fsys is downloaded under the name above.
